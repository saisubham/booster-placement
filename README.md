# booster-placement

To compile the program type the following command
```
$ cd /download_path/booster-placement/
$ make all
```

To run the program
```
$ ./tvs input_file.txt
```

Some sample input files are provided in /test folder. To see how they work type
```
$ make all
$ ./tvs ./test/input2.txt
```

To clean the directory
```
$ make clean
```
